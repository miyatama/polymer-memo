
Polymer-memo
====

Overview
todo web service that use polymer

## Description
can regist , remove todo. 

## Requirement
node.js
bower

## Install
bower update

edit ./service/config.php

create table in db server
create table t_material_memo(
  id auto_increment
  ,title varchar(100)
  ,memo varchar(200)
  ,delflg int
)

upload web server and access
